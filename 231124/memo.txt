WWW (World Wide Web)
클라이언트 <-> 서버
서버측면에서 봤을때 클라이언트와의 연결은 stateleass(연결이 되어있지 않다)!

<쿠키>
: 서버(웹사이트)가 저장함.
: 도메인별로 저장
: 사용자의 예전 검색기록 등 사용자와 관련된 정보를 저장함
: 저장공간 (5kbyte)
: 쿠키의 내용은 해당 도메인으로 보내는 모든 요청에 첨부함!

<로컬스토리지 : Local Storage>
: 클라이언트(자바스크립트)가 저장함
: key와 value로 구성됨(=자바스크립트 객체와 유사)
: 모든 종류의 정보를 저장할 수 있음
: 저장공간 무제한(브라우저에서 세팅가능)
: 저장내용은 클라이언트가 자신의 목적으로 사용함. (서버에 보낼 수도 있고 아닐 수도 있음.)

<세션스토리지>
: 쿠키 + 로컬스토리지의 특정을 가짐
: 서버와 클라이언트 모두 사용
: 주로 보안목적으로 사용되는 경우가 많음.
